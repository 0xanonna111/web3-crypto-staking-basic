# Web3 Crypto Staking Basic

This project helps beginners learn how crypto staking works using blockchain.

You will learn:
- What staking is
- How to connect a wallet
- How to stake tokens
- How to claim rewards
- How to read staking data

## Requirements
- Node.js installed
- MetaMask browser extension
- Basic JavaScript knowledge

## Setup

1. Clone the repo
git clone https://github.com/yourusername/web3-crypto-staking-basic.git

2. Install packages
npm install

3. Run the app
npm start

4. Open browser
http://localhost:3000

## Project Structure

- index.html = Main webpage
- app.js = Frontend logic
- staking.js = Staking contract functions
- web3.js = Blockchain connection
- deploy.js = Contract deploy script
- config.js = Network settings
- server.js = Local server
- package.json = Project info
- utils.js = Helper functions
- .env.example = Environment example

## Features
- Connect wallet
- Stake tokens
- Claim rewards
- Show staked amount
- Simple staking dashboard

## Learning Goal
This project explains how staking systems work in a simple way.
